新旧模式对比
评论 → 讨论，两代产品的核心差异一览
💬 经典评论模式（升级前）
📝 名称 评论 / 回复
🔗 结构 混排线性，所有讨论堆叠在一起
🧵 话题性 无，一个 Event 下讨论混杂
📌 转任务 只能针对整条评论，无法精准拆解
👀 可见性 所有回复默认全部展开
⚡ @提及 提及即通知，无法区分范围
🧵 Thread 讨论模式（升级后）
📝 名称 讨论 / 回复
🔗 结构 每个讨论独立成串，互不干扰
🧵 话题性 强，每个 Thread 有独立标题
📌 转任务 可针对单个讨论精准创建子任务
👀 可见性 默认折叠，点击展开，沉浸式阅读
⚡ @提及 提及 + 范围标注，精准触达
